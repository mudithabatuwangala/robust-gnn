# robust-gnn

Graph Neural Network implementation (currently optimized for CPU).

## Setup Instructions

1. **Clone the Repo**: Run `git clone https://github.com/mudithabatuwangala/robust-gnn.git` and enter the folder.
2. **Create Environment**: Run `python -m venv venv` to create your local environment.
3. **Activate Environment**:
   - Windows: `.\venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. **Install Packages**: Run `pip install -r requirements.txt` to install Torch and PyG.
5. **Run Code**: Run `python main.py` to verify the installation.
